O PROJETO

O que é o Educodar?

O Educodar é uma iniciativa sem fins lucrativos, que foi criada pelos funcionários da empresa ThoughtWorks e tem o objetivo de apresentar os conceitos de programação para adolescentes de escolas públicas.

Turmas

O projeto existe desde 2014 e já atingiu mais de 70 adolescentes acima dos 14 anos. As turmas já aconteceram em Porto Alegre e Belo Horizonte, ensinando para os alunos o básico de lógica de programação.

Apoiadores do Projeto

Apoiadores: ThoughtWorks, TECNOPUC, Alura, Escola do SEBRAE

Já apoiaram: PUCRS
